# 项目管理阶段标签扩展说明

## 新增内容

- 在项目管理工作台左侧阶段标签中新增 `投资估算`。
- 在项目管理工作台左侧阶段标签中新增 `初设概算`。

## 实现功能

- `投资估算` 与 `施工图预算` 使用完全相同的表格列、树表数据、右侧基本信息、审核意见内容。
- `初设概算` 与 `施工图预算` 使用完全相同的表格列、树表数据、右侧基本信息、审核意见内容。
- 两个新增阶段沿用现有项目管理页功能，包括行选中、新建项目、新建造价文件、双击第二层造价文件的 `计价库` 单元格进入预算书工作区、返回后恢复页面状态等。

## 对应文件

- `src/views/cost/project-management/types.ts`
  - 扩展 `WorkbenchStageKey`，新增 `investmentEstimate`、`preliminaryEstimate` 两个阶段 key。
- `src/views/cost/project-management/mock.ts`
  - 在 `stageOptions` 中新增两个左侧阶段标签。
  - 抽出 `budgetStageColumns` 复用施工图预算表格列配置。
  - 在 `stageConfigs` 中新增 `投资估算`、`初设概算` 两个阶段配置，并复用 `施工图预算` 的行数据与功能配置。
- `src/views/cost/project-management/index.vue`
  - 调整左侧阶段按钮最小高度，避免阶段数量增加到 6 个后在常见视口下被裁切。
  - 将阶段按钮的竖排文字规则收敛到内部文本上，并禁止四字标签自动换列，保证 `投资估算`、`初设概算`、`工程结算`、`造价指标` 等四字内容单列展示。
  - 新增项目管理主表右键菜单，菜单内容按参考图分组展示。
  - 新增右键菜单的 `新建` 二级菜单，包含 `建设项目`、`单项工程`、`造价文件`。
  - 新增右键菜单的 `显示设置` 二级菜单，包含 `设置颜色`、`锁定此列`、`撤销列锁定`。
  - 右键菜单使用 `Teleport` 挂载到页面 body，避免被表格容器裁切，并根据菜单实际高度自动调整显示位置。
  - 右键菜单的父级项支持点击展开子菜单，同时保留鼠标悬停展开效果。
  - 调整 `显示设置` 二级菜单为底部对齐弹出，避免靠近页面底部时被遮挡。
