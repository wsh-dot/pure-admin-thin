# 措施项目明细标签联动优化说明

## 新增内容

- 优化清单计价模式下 `措施项目` 主标签页的上方主表数据。
- `措施项目` 主表新增 `工程`、`清单`、`分部` 三类行，用于模拟截图中的行层级。
- 新增 `措施项目` 行级明细状态：
  - `清单` 行复用分部分项清单行的下方标签体系。
  - `分部` 行展示 `人材机`、`主材设备` 两个下方标签。

## 实现功能

- 点击 `措施项目` 中标识为 `清单` 的行时，下方标签页和对应表格字段与 `分部分项` 的清单行一致。
- 点击标识为 `分部` 的行时，下方默认展示 `人材机` 表格字段。
- 在 `分部` 行下切换到 `主材设备` 标签时，展示主材设备表格字段。
- `措施项目` 与 `分部分项` 共用现有主表 + 明细区框架，避免重复维护两套近似界面。
