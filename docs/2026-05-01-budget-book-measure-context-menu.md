# 预算书措施项目右键菜单说明

## 新增内容

- 在预算书页面补全措施项目主表右键菜单，菜单按截图包含选择、增加、剪切、复制、粘贴、删除、清单工程内容设置、工料机价格、保存到、导入/导出、批量、定额、检查、显示内容、计算精度设置等入口。
- 为选择、增加、工料机价格、保存到、导入/导出、批量、定额、检查、显示内容、计算精度设置补充对应的二级菜单内容。
- 右键菜单现在支持在清单计价的措施项目标签页展示，同时保留原预算书主表的右键菜单展示能力。
- 菜单打开和悬停展开二级面板时，会重新计算主菜单位置，为右侧详细面板预留空间，降低二级面板被浏览器边界遮挡的风险。

## 实现功能

- 右键措施项目表格区域时展示完整菜单。
- 悬停或点击带箭头的菜单项时展示对应详细面板。
- 详细面板支持向下或向上对齐，靠近底部的菜单项不会向屏幕外继续展开。
- 批量菜单中的“设置取费类别”支持再次显示下级箭头与下级内容，满足截图中的多级提示样式。

## 对应文件

- `src/views/cost/project-management/budget-book.vue`
  - `budgetContextMenuGroups`：维护右键菜单和详细面板内容。
  - `handleBudgetContextMenu`：控制预算书/措施项目主表右键菜单显示。
  - `adjustBudgetContextMenuPosition`：控制菜单位置，保证详细面板尽量可见。
  - 右键菜单模板：渲染一级、二级和局部三级菜单结构。
