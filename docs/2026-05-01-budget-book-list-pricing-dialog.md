# 清单计价弹窗说明

## 新增内容

- 清单计价分部分项标签页右侧快捷入口中，将原来的“定额计价”文案改为“清单计价”。
- 点击“清单计价”后打开清单计价弹窗，默认选中“清单模板”标签页。
- 清单计价弹窗新增“清单模板”标签页，与原有“定额库 / 工料机库 / 指标库”并列。
- 清单模板页展示模板选择、添加选中、展开、收缩按钮，以及清单计算规则树。

## 实现功能

- 定额计价模式下仍保留原“定额计价”入口，打开时默认进入定额库。
- 清单计价分部分项模式下显示“清单计价”，打开时默认进入清单模板。
- 在弹窗内切换到定额库、工料机库、指标库时，展示对应的树、表格和说明内容。
- 清单模板页使用独立样式，保证内容区域完整可见。

## 对应文件

- `src/views/cost/project-management/budget-book.vue`
  - `PricingLibraryTabKey`：新增 `template` 类型。
  - `pricingLibraryTabs`：新增“清单模板”标签页。
  - `templateTreeNodes`：维护清单模板树数据。
  - `currentShortcutLinks`：按当前模式切换快捷入口文案。
  - `handleToolbarAction`：区分“定额计价”和“清单计价”弹窗默认标签。
  - 清单计价弹窗模板和样式：渲染清单模板、定额库、工料机库、指标库。
