# 清单计价主标签展示修正说明

## 新增内容

- 修正预算书切换到清单计价后的顶部主标签体系。
- 新增清单计价模式专用主标签：
  - 基本信息
  - 分部分项
  - 措施项目
  - 其他项目
  - 工料机汇总
  - 取费程序
  - 调价
  - 报表
- 新增 `措施项目` 主表 mock 数据和字段。
- 新增 `其他项目` 左侧清单树和右侧其他项目清单表格 mock 数据。

## 实现功能

- 点击 `切换清单计价` 后，默认选中顶部主标签 `分部分项`。
- `分部分项` 标签展示清单计价主表和下方人材机明细区。
- 点击 `措施项目` 标签后，展示措施项目表格，下方保留人材机明细区。
- 点击 `其他项目` 标签后，展示左侧其他项目清单树和右侧其他项目明细表。
- 清单计价模式下仍可点击 `取费程序` 主标签进入已有取费程序工作区。
