# 预算书定额计价弹窗说明

## 新增内容

- 在预算书页面新增“定额计价”弹窗。
- 弹窗包含三个标签页：定额库、工料机库、指标库。
- 弹窗顶部提供行业/年度筛选区，并展示当前库名称。
- 弹窗左侧提供章节查询/条件查询切换外观和章节树。
- 弹窗右侧根据当前标签页展示对应表格数据。
- 定额库和指标库下方新增说明区域，展示定额说明/指标说明内容。

## 实现功能

- 点击预算书页签右上角“定额计价”按钮时，打开定额计价弹窗。
- 默认打开“定额库”标签页。
- 点击“工料机库”时，切换为工料机分类树和工料机表格。
- 点击“指标库”时，切换为指标章节树、指标表格和指标说明。
- 弹窗布局、表格列、左侧树、选中态和底部说明区按参考图 1、2、3 的结构实现。
