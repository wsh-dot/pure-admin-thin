# 预算书右键菜单功能说明

## 新增内容

- 在预算书工作区新增右键菜单。
- 右键菜单主菜单按参考图展示 `选择`、`增加`、`剪切`、`复制`、`粘贴`、`删除`、`关联定额`、`保存到`、`导入/导出`、`批量`、`定额`、`检查`、`显示设置` 等操作。
- 支持点击或悬停展开二级菜单。

## 二级菜单内容

- `选择`：定额、指标、工料机。
- `增加`：分部、定额、指标、计算项、其他费用项、人工、材料、机械、主材、设备。
- `关联定额`：当前定额、全部专业。
- `保存到`：我的定额库、我的工料机库、我的分项模板。
- `导入/导出`：块导入、块导出、块导入3.0文件、导入工程量计算表、导入算量清单。
- `批量`：工程量乘系数、设置子目增加费、取消子目增加费、取消选中定额增加费、恢复选中定额增加费。
- `定额`：刷新定额、取消选中定额调整、取消所有定额调整、定额替换。
- `检查`：检查工程量为负、检查人材机消耗为负、检查人材机消耗为0、删除定额(数量\*单价)工程量为0、定额工程量与主材设备工程量是否匹配、定额取费合计占比总造价比例。
- `显示设置`：设置颜色、列显示/隐藏、显示审核标记。

## 对应文件

- `src/views/cost/project-management/budget-book.vue`
  - 新增预算书右键菜单数据结构、显示状态、点击/悬停展开逻辑。
  - 在预算书主工作区绑定右键事件。
  - 通过 `Teleport` 将菜单挂载到页面 body，避免被表格区域裁切。
  - 对靠近页面底部的二级菜单使用底部对齐方式，避免内容被遮挡。
