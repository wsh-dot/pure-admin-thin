# 预算书取费程序工作区与叶子节点标签修正说明

## 本次新增与修正内容

- 修正预算书主树表选中最底层叶子节点时的底部标签页数量。
- 叶子节点底部标签页现在只保留原型要求的 3 个：
  - `主材设备`
  - `工程量计算式`
  - `取费程序`
- 保留父节点和定额节点各自独立的标签集合，不再统一按“非父节点显示 6 个”处理。

- 新增预算书顶部主标签 `取费程序` 的独立工作区原型实现。
- 该工作区采用三块布局：
  - 左上：取费类别列表
  - 左下：设置项目 / 设置值表格
  - 右侧：取费程序树表
- 新增两条可拖拽分隔线：
  - 左右分隔线：可左右拖动，调整左侧分类区与右侧树表宽度
  - 上下分隔线：可上下拖动，调整左上分类区与左下设置区高度
- 点击左侧不同取费类别时，右侧树表与左下设置表会同步切换 mock 数据。
