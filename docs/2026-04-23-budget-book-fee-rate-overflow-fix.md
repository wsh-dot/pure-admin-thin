# 费率表收起不可见修复说明

## 本次修复内容

- 修复 `取费程序 -> 显示费率` 场景下，费率表已经被拖到最小高度后，视觉上仍然残留显示的问题。
- 为费率分区和通用明细表容器补充 `overflow: hidden`，确保当面板高度被压到 `0` 时，内部表格内容不会继续溢出显示。

- 此前拖拽高度实际上已经能缩到 `0`，但承载 `vxe-table` 的 grid 子项默认允许内容外溢。
- 结果就是下方面板虽然在布局上被压缩了，表格内容仍然会从容器外露出来，看起来像“无法完全收起”。
