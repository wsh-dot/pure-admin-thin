# 预算书计价模式切换功能说明

## 新增内容

- 将预算书工具栏原 `局部汇总` 按钮改为计价模式切换入口。
- 按钮默认显示 `切换清单计价`，点击后进入清单计价模式并显示为 `切换定额计价`，再次点击回到定额计价模式。
- 新增清单计价模式主表字段和 mock 行数据，用于展示工程、分部、清单、定额、工作内容等行。
- 新增清单行、定额行、工程行对应的底部标签页展示逻辑。

## 实现功能

- 清单计价模式下，主表展示完整字段：编号、标识、名称、项目特征、工作内容、单位、工程量表达式、工程量、单价、合价、取费类别、人工合价、材料合价、机械合价、主材合价、设备合价、管理费合价、利润合价、备注。
- 点击标识为 `清单` 的行时，下方展示 `人材机`、`主材设备`、`特征及内容`、`工程量计算式`、`取费程序`、`清单计价规则` 标签页。
- 点击标识为 `定额` 的行时，下方展示 `人材机`、`主材设备`、`子目增加费`、`工程量计算式`、`取费程序`、`定额说明` 标签页。
- 点击标识为 `工程`、`分部` 的行时，下方仅展示 `人材机` 标签页。
- `特征及内容` 标签页新增左右双表展示，左侧为项目特征，右侧为工作内容。
- `清单计价规则` 和 `定额说明` 标签页新增文本内容展示。

  - 新增清单计价模式主表字段、主表 mock 数据和行级底部明细状态。
- `src/views/cost/project-management/budgetBookTypes.ts`
