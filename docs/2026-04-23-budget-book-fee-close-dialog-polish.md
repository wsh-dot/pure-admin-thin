# 预算书取费程序关闭入口与弹窗优化

## 本次新增与调整

- 将 `取费程序` 主工作区的关闭按钮从上方主标签行移出，改为放到右侧模板工具栏最右侧，与 `我的取费模板` 保持同一行。
- 保留其他主标签场景下的关闭入口，避免 `预算书` 等页签失去关闭能力。
- 将关闭确认弹窗改为更简洁的 `Element Plus el-dialog` 信息式布局，增加标题、风险提示说明和更清晰的操作按钮。

## 实现效果

- 当顶部主标签切换到 `取费程序` 时：
  - 关闭按钮显示在模板工具栏最右侧。
  - 上方主标签栏不再重复显示叉叉按钮。
- 点击关闭按钮后：
  - 弹出更清晰的确认对话框。
  - 支持 `保存并关闭`、`直接关闭`、`取消` 三种操作。
